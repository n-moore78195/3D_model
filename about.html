Interactive Can Modeling Showcase

This project upgrades classroom can models with improved geometry behavior, material readability, lighting and camera setup, and JavaScript-driven UI interactions such as wireframe toggles, animation triggers, and responsive model switching.

Originality Statement

Source 3D assets were remixed with custom interaction code and UI integration. The final app workflow, dynamic action system, scene behavior, and presentation layout were developed as original coursework implementation.

Site Map

  • Home (`index.html`): 3D viewer, model gallery, wireframe/lighting/tools, and animation actions.
