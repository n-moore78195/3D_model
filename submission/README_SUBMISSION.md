# Submission Package

This folder is prepared for coursework submission.

## Structure

- `original_3d_files/`
  - editable 3D source files (`.blend`) and export script
  - `final_glb_blends/` contains one `.blend` per final GLB model
- `docs/`
  - code documentation and source mapping

## Recommended Attachments

Along with this folder, include runtime web app files:

- `index.html`
- `styles.css`
- `app.js`
- `about.html`
- `models/` (contains the 6 final GLB files; `Coca_Cola.blend` is retained for export workflow)
